Description

The term ’’hypebeast’’ - a person who follows a trend to be cool or in style (bustle.com), has been used very frequently throughout the current young generation. Keeping up with the lastest fashion, sneakers, music and sports is a must for the youth and for me. My objective for this website was to provide the latest ’’hype’’ fashion, sneakers, music and sports feeds.

Development

Front-End

For the front-end of my project, I took an semantic appoarch to the HTML for easier readability. Also, I used viewport to develop an responsive look to the website. Lastly, Javascript was used to handle the user forms.

Developed using HTML, CSS, & Javascript

Back-End

For the back-end of my project, I used PHP to help link my header/footer files on every page. Along with, inputing data into my database from the user forms. Also, I implemented a RSS parser that pulls RSS feeds from websites. Lastly, php was used to start sessions and end them whenever a user logs on or logs off.

Developed using PHP

Database

For the database part of my project, I used MySQL. Along with phpmyadmin, which was extremely easy to use and access my database. I implemented a SQL script to take in the users, email, username, and password information to register. The registerd data is secured and is used to allow users to comment on posts.

Developed using MySQL
